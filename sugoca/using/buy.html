【승차권 구입】

  • SUGOCA카드 사용 가능 구역 내에서 승차하여 사용 가능 구역 밖의 역에서 하차할 경우에는 사전에 자동발매기에서 전구간의 승차권을 구입하십시오.
  • 매표소에서 승차권을 구입할 경우에는 SUGOCA카드 내의 잔액을 사용할 수 없습니다. 현금 또는 신용카드로 구입하십시오.
  • 승차권을 구입할 경우에는 여러장의 SUGOCA카드를 사용할 수 없습니다.
  • 승차권을 구입할 경우, SUGOCA카드를 사용하여 동일한 구간을 승차했을 경우와 운임이 다를 수도 있습니다. (SUGOCA카드 사용 가능 구역)

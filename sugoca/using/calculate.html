    • 승차권 구입・정산 방법

    • 환승 개찰구의 이용에 관하여
    • 전자화폐
    • Q&A・주의 사항
    • 후쿠오카 교통 안내도

하차역의 자동개찰기를 통해 나갈 때 충전(입금)되어 있는 카드 내의 잔액에서 이용 구간의 운임요금이 지불됩니다.

  • SUGOCA카드 내의 잔액을 사용하여 승차할 때, 잔액이 10엔 미만인 경우에는 개찰기를 통과할 수 없습니다. 자동발매기 등에서 충전(입금)한 뒤에 이용하십시오.
SUGOCA카드 내의 잔액을 이용하여 카드 사용 가능한 구역(아래 그림)에서 승차할 경우 승차 경로에 관계 없이 가장 저렴한 운임이 소요됩니다. (승차권을 구입하는 것보다 싼 경우가 있습니다.)
●오리오 역에서 환승할 경우의 주의사항
카드 내의 잔액을 이용하여 오리오 역의 본역(1~5번 타는 곳)과 타카미 입구(6•7번 타는 곳)사이에서 카고시마 본선과 후쿠호쿠유타카선을 환승할 경우에는 자동개찰기에 SUGOCA카드를 터치하지 마십시오. 자동개찰기에 터치하게 되면, 승차역에서 오리오 역까지의 운임을 일단 정산하기 때문에, 오리오 역에서부터 최종 하차역까지의 운임을 새롭게 계산하게 됩니다. 이 경우 실제의 운임보다 비싸질 수 있습니다.
●카라츠선(카라츠~쿠보타~사가)을 경유할 때는 SUGOCA카드를 이용할 수 없습니다.
카카라츠선(카라츠~쿠보타~사가)을 경유하여 카라츠선•치쿠히선의 각 역(메이노하마~니시 카라츠)에서 SUGOCA카드 사용 가능 구역 내의 각 역에 걸친 구간을 이용할 경우에는 SUGOCA카드로 승차할 수 없습니다. 사전에 승차 경로에 따라 승차권을 구입하십시오.
